## Про проєкт

Лендінг-сторінка для рок-гурту «Грим та Грім», створена як навчальний проєкт. Містить інформацію про гурт, найближчі концерти, учасників, галерею та форму зворотного зв'язку.

## Функціональність

- **Навігація** — фіксований хедер з плавним скролом до секцій; адаптивне бургер-меню (< 700px) зі зміною іконки burger ↔ close; scroll spy — підсвічування активного пункту меню
- **Концерти** — таблиця найближчих концертів з кнопками замовлення квитків
- **Pop-up вікна** — модальні вікна для: замовлення квитків (з формою), карток учасників, збільшення фото галереї, соціальних мереж
- **Форми** — валідація полів (ім'я, email, повідомлення); імітація GET запиту через `history.pushState` при успішній відправці; повідомлення про успіх
- **Анімації** — поява елементів при скролі через `IntersectionObserver`; кнопка "вгору" при скролі > 400px; parallax ефект на hero-секції (тільки десктоп)
- **Галерея** — сітка фото з hover-ефектом; відкриття в повноекранному pop-up
- **Адаптивність** — breakpoints: 700px (мобільне меню), 900px (футер), fluid-сітки для карток та галереї
